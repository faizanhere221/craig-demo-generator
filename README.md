# Demo Site Generator - Optimo Agency

Generate professional demo websites for prospects in 30 seconds!

## Features

- ✅ Simple VA-friendly form
- ✅ 5 niche templates (Roofers, Painters, Builders, Renovations, Landscapers)
- ✅ Auto-deploy to Vercel
- ✅ Mobile responsive sites
- ✅ Professional dark theme design

## Setup

### 1. Install Dependencies

```bash
npm install
```

### 2. Get Vercel Token

1. Go to https://vercel.com/account/tokens
2. Create new token with name "Demo Generator"
3. Copy the token

### 3. Configure Environment

Create `.env.local` file:

```
VERCEL_TOKEN=your_vercel_token_here
```

### 4. Run Locally

```bash
npm run dev
```

Open http://localhost:3000

### 5. Deploy to Vercel

```bash
vercel --prod
```

## Usage

1. Open the generator form
2. Enter:
   - Company Name (e.g., "Smith Roofing")
   - Phone (e.g., "0412 345 678")
   - Niche (select from dropdown)
   - Location (e.g., "Sydney, NSW")
3. Click "Generate Demo Site"
4. Get live URL in ~30 seconds!

## Niches Supported

4. **Renovation** - Home renovation
